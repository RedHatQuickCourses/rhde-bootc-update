= Apply Updates to Disconnected Image Mode Systems

Goal::

Perform disconnected updates of RHEL for edge devices with image mode for RHEL.

WARNING: Not started yet

== Introduction

This chapter describes a worflow for applying updates updates to image mode to disconnected systems, by retrieving the new bootc container image from removable media.

While this worflow is not directly supported by the bootc utility, its implementation requires just a few commands using RHEL container tools such as Skopeo, and could be automated by using standard RHEL tools, such as Systemd.

Your image mode systems may not be fully disconnected, but require a disconnected update workflow because the cannot reliably download multiple GB of bootc container images over the network.
