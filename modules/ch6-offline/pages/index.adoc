= Apply Updates to Disconnected Image Mode Systems

Goal::

Perform disconnected updates of RHEL for edge devices with image mode for RHEL.

== Introduction

This chapter describes a workflow for applying updates to image mode to disconnected systems, by retrieving the new bootc container image from removable media.

While this workflow is not directly supported by the bootc utility, its implementation requires just a few commands using RHEL container tools such as Skopeo, and could be automated by using standard RHEL tools, such as Systemd.

Even if your edge devices are fully disconnected, they might benefit from a disconnected update workflow if they cannot reliably download multiple GB of bootc container images over the network.
