= Undo Updates on Image Mode Systems

Goal::

Undo updates of RHEL for edge devices with image mode for RHEL.

== Introduction

This chapter describes the use of the bootc utility to roll back the active system image to a previous bootc container image, as in returning a system to a previously known good configuration.

Image mode systems never discard the last successfully booted system image. It stays in the root disk, and you can make it again the active image.
